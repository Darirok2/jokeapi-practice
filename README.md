# JokeAPI Client

**Учебный проект по практике "Основы Git и работа со сторонними REST API"**

Простое консольное приложение на Python, которое получает шутки через открытый API — [JokeAPI v2](https://v2.jokeapi.dev/).

## Возможности

- Получение случайной шутки
- Выбор категории шутки (Programming, Dark, Christmas и другие)
- Фильтрация нежелательного контента (nsfw, religious, political и т.д.)
- Сохранение результата запроса в JSON-файл
- Удобный интерактивный интерфейс в терминале

## Доступные категории

- `Any` — любая шутка
- `Programming` — программистские шутки
- `Miscellaneous` — разное
- `Dark` — чёрный юмор
- `Pun` — каламбуры
- `Spooky` — страшилки
- `Christmas` — новогодние шутки

## Как запустить

Рекомегдуется запускать в виртуальном окружении
```bash
pip install -r requirements.txt
```
```bash
python main.py
```

Это учебный проект, не предназначенный для коммерции!!!
